## Run on host

### Installation
```bash
git clone https://github.com/epanchee/link_watcher
pip install pipenv
pipenv install
```

### Start
```bash
pipenv shell
python run.py -c config/schema.yaml
```

## Run using docker-compose

### Installation
```bash
git clone https://github.com/epanchee/link_watcher
```

### Start
```bash
docker-compose up -d
```
